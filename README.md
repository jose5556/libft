# Libft

This is the libft project, completed during my studies at School 42. The aim of this project is to create a library of custom implementations of standard C library functions.

## Overview

In this project, I have implemented various functions commonly found in the standard C library (<string.h>, <stdlib.h>, etc.) from scratch. These functions include string manipulation, memory allocation and others.

For more detailed information, please take a look at the subject file located alongside the other project files.

Feel free to read the libft.h header file for all the implemented functions.
